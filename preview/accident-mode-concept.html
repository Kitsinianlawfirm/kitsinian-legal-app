Implementation Notes

  • Entry Point: "I Was In An Accident" emergency button on home screen, or triggered from notification
  • GPS & Weather: Captured automatically using device location and weather API
  • OCR: Use Vision framework (iOS) to extract text from license/insurance photos
  • Voice Recording: Transcribed to text for case file, original audio preserved
  • Retainer: Digital signature via SignatureView, legally binding e-signature
  • Offline Mode: All data saved locally, synced when connection available
  • Time Sensitivity: Show "Evidence collected 3 mins after accident" in case file
